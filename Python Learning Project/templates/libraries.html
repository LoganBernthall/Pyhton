    TensorFlow:


    TensorFlow is a free, open-source software library for machine learning and AI. It can be used across a plethora of tasks such as training and inference of deep neural networks.


    TensorFlow website:


    Scikit-learn:


    Scikit-learn is a free software machine learning library for Python. Scikit-learn features classification, regression and clustering algorithms as well as support-vector machines, random forests, gradient boosting, K-means and much more.


    Scikit-learn website:

    Numpy is the fundamental package for scientific computing in Python. It is a Python library that provides multidimensional array object.


    Numpy website:

    PyTorch is a machine learning framework based on the Torch library, used for applications such as computer vision and natural language processing.

    LightGBM is a gradient boosting framework that uses tree- based learning algorithms. It is designed to be distributed and efficient.
